# Review Notes

Published review notes for the screenplay tool live here.

- `published.json` is loaded by GitHub Pages so every visitor sees the same
  published notes.
- Draft notes are still saved in each reviewer's browser while they are writing.
- To publish a note, use **Publish to GitHub** in the shot review panel. The site
  opens a structured GitHub issue, and the `Publish review note` workflow commits
  the note into `published.json`.

Attachments selected in the browser stay in the local draft note. Published notes
include attachment metadata only, so the public notes file stays small and safe for
GitHub Pages.
